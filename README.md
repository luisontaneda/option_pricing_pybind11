# option_pricing_pybind11

Option pricing library using pybind11.

## Types of Option Contracts

- **European**:

  - Vanilla Options
  - Exotic Options:
    - Asian

- **American**:
  - Vanilla Options

## Methods used:

- Montecarlo
- COS Method
